# **App Name**: LeadWise

## Core Features:

- Leadership Style Assessment: Interactive quiz to assess the user's leadership style and identify areas for improvement.
- Personalized Feedback: Provides immediate, personalized recommendations based on the assessment results.
- Situation Simulations: Presents 3-5 common leadership scenarios with multiple-choice responses and feedback on best practices. Each one includes a reasoning AI tool for optimal practices.
- Dynamic Book Recommendations: Recommends books based on identified weaknesses, providing links to purchase on Books.com.tw. Books are determined based on answers to the 'Leadership Style Assessment' tool.

## Style Guidelines:

- Primary color: Deep blue (#3498DB) for trustworthiness and reliability.
- Background color: Light blue (#EBF5FB), subtly tinted and desaturated from the primary, to provide a calm background
- Accent color: Teal (#45B39D), is used for highlighting interactive elements and calls to action. It sits approximately 30 degrees from the primary hue and stands out well against both the light background and the primary color.
- Headline font: 'Playfair', a modern sans-serif with an elegant, high-end feel, used for titles and short texts.
- Body font: 'PT Sans', a humanist sans-serif used for body and longer texts.
- Use clean, professional icons representing leadership skills.
- Clean, card-based layout for easy navigation and clear information hierarchy.
- Subtle transitions and feedback animations for a polished user experience.
